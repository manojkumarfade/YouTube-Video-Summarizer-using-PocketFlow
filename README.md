# YouTube Video Summarization and Q&A System

## Overview
This project builds an AI-powered system that extracts transcripts from YouTube videos and generates concise, structured summaries using Large Language Models (LLMs).

It also supports follow-up Q&A on video content for deeper understanding.

## Key Features
- Automatic transcript extraction
- LLM-based summarization
- Interactive question answering
- Prompt-engineered responses for accuracy

## Tech Stack
- Python
- LLM API (OpenAI / Gemini)
- PocketFlow
- NLP techniques
- Google Colab

## How It Works
1. Extracts video transcript
2. Cleans and segments the content
3. Generates summaries using an LLM
4. Allows users to ask follow-up questions on the video

## Use Cases
- Educational video summarization
- Research content digestion
- Knowledge extraction from long videos
